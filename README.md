# Kali Linux on PinePhone Pro

## Objectif
Déploiement d'un environnement Kali Linux complet sur PinePhone Pro
en dual boot, pour un usage mobile discret des outils de cybersécurité.

## Étapes réalisées
- Recherche et sélection du matériel (PinePhone Pro — ARM64)
- Étude de l'architecture et du bootloader (U-Boot)
- Partitionnement et configuration dual boot
- Portage et installation de Kali Linux ARM64
- Configuration complète de l'environnement (outils, réseau, OPSEC)

## Pourquoi le PinePhone Pro
- Hardware ouvert et documenté
- Killswitches matériels (micro, caméra, WiFi, LTE)
- Aucun firmware propriétaire — contrôle total de la stack
- Discrétion opérationnelle maximale
